HOW TO USE

-+- MAIN MENU -+-

FOR STARTING CLIENT
1. Enter IP Address and PORT #.
2. Select deck or add own deck (refer to deck rules for custom decks).
3. Select 'Load Deck'.
4. Once server is setup, select 'Start Client'.

FOR STARTING SERVER (requires port fowarding)
1. Enter IP Address and PORT #.
2. Select 'Start Server'.

-+- CUSTOM DECK RULES -+-

1. Go to 'https://www.carddweeb.com/DeckBuilder'.
2. Create deck: Must not include newest 2026 expansion cards, last usable card index is 568.
   To check look at the card # in the url of the card using the 'Card Database' tab.
3. Select 'Validate Deck', if valid, select 'Copy decklist to clipboard'
4. Return to game, paste value into custom deck input field.
5. Select 'Load Deck', hero name will appear in log in top right if successful.
